# Tic Tac Toe ゲーム実装

## 背景/目的

Web ブラウザで動作する ○✕ゲーム（Tic Tac Toe）を、Vanilla JavaScript で実装します。シングルプレイヤー モードで Human が AI 対戦相手と対戦する形式とします。基本的なゲームロジック実装から開始し、将来的には複数ゲームモード拡張を考慮した設計とします。

## 技術情報・実装方針

### テクノロジースタック
- **言語**: JavaScript (ES6+)
- **ランタイム**: Node.js 18+ (ローカル開発), Web ブラウザ（実行）
- **テスト**: Jest v29+
- **ビルドツール**: なし（Vanilla 構成）
- **サーバー**: http-server (npm run dev 用)

### アルゴリズム
- **ミニマックス**: ゲーム木探索で最適手を決定
- **評価関数**: 勝利=+10, 敗北=-10, 中立=0
- **最適化**: アルファベータ枝刈り（オプション）

### 画面レイアウト（HTML 構成）

## スコア・統計情報について

### 連勝数
- ゲームリセット後、Human が勝利するたびに +1
- Human が負けた時点でリセット（0に戻す）
- UI に「Win Count: X」として表示
- LocalStorage 保存は Phase 2 で実装
