# TIC TAC TOE 実装ドキュメント

## ゲームルール

- 3x3 の盤面で Human（`○`）と AI（`✕`）が交互に手を打ちます。
- 同じ行・列・斜めのいずれかで 3 つ揃えた側が勝利です。
- 9 マスすべて埋まり、勝者がいない場合は引き分けです。
- 勝敗が確定した時点で盤面操作は無効化されます。

## ミニマックス AI の説明

- AI は `src/gameLogic.js` の `getBestMove` で手を選択します。
- 探索はミニマックス法を使用し、終局までの分岐を再帰的に評価します。
- 評価関数:
  - AI 勝利: `10 - depth`（早く勝つほど高得点）
  - Human 勝利: `depth - 10`（遅く負けるほど高得点）
  - 引き分け: `0`
- 既に終局している盤面では `null` を返します。
- AI 思考時間を計測して UI に表示し、`500ms` 以下かどうかを可視化します。

## ファイル構成 / 責務

- `src/gameLogic.js`
  - 初期盤面生成、着手適用、勝敗判定、勝利ライン取得、ミニマックス探索
- `src/controller.js`
  - ゲーム進行制御（Human→AI の手番遷移、勝敗時の Win Count 更新、リセット）
- `src/ui.js`
  - DOM の生成・再描画、セルの有効/無効制御、勝利演出クラス付与
- `styles/main.css`
  - レイアウト、ダークモード、ホバー/クリック/勝利時の視覚効果
- `tests/gameLogic.test.js`
  - ロジック単体テスト（勝敗判定、無効手、最善手選択、性能）
- `tests/controller.test.js`
  - UI/進行の結合テスト（描画、手番、Win Count、AI 思考時間表示）
