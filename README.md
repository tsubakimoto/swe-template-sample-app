# ○✕ゲーム (Tic-Tac-Toe)

Vanilla JavaScript で作成した、1人用の ○✕ゲームです。  
プレイヤーは **○ (Human)**、AI は **✕ (Minimax)** で対戦します。

## セットアップ

```bash
npm install
```

## 実行

```bash
npm run dev
```

## テスト

```bash
npm test
```

## ビルド

```bash
npm run build
```

## 主な仕様

- 3x3 盤面
- 勝敗 / 引き分け判定
- リセット機能
- AI は minimax による最適手を選択
- AI 思考時間表示
